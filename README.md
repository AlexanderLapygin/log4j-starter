- [Java 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

---

## Build

```sh
mvnw clean package
```

## Run

```sh
java -cp target/hello-world-1.0-SNAPSHOT.jar com.example.App
```

Watch 'Hello World!' in the console.
